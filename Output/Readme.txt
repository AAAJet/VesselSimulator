INSTALLATION
============

1. Copy the intact 'Engineer' folder into the 'GameData' folder within your Kerbal Space Program installation directory.

2. Start the game and play.

3. Attach one of the Engineer parts found under the science tab to you ship.
    - ER7500 includes build + flight engineer.
    - Chip #1 includes build + flight engineer.
    - Chip #2 includes only the build engineer.
